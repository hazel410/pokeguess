# Pokeguess

- A 20-question-style game centered around pokemon!
- Think of any pokemon and it will guess it in 12 questions or less!
- Theres also a hidden button in the bottom left to spawn in little creatures :)
- This was my project to learn python, and OOP-- so its not the greatest, but ignoring the bad code its pretty cool!
## How to Play
1) install the required packages in [requirements.txt](requirements.txt)
2) clone the repo and run the python script! (I ran it in vscode)
